# 🎯 Мультиплеєрна вікторина

Простий додаток вікторини з підтримкою кількох гравців у реальному часі.

## Технологічний стек

- **Сервер:** Node.js, Express, Socket.io
- **Клієнт:** React (create-react-app), Socket.io-client
- **Дані:** питання зберігаються у масиві на сервері

## Структура проєкту

```
quizgame/
├── server/           # Серверна частина
│   ├── server.js     # Основний сервер
│   ├── questions.js  # Масив питань
│   └── package.json
├── client/           # Клієнтська частина (React)
│   ├── public/
│   │   └── assets/   # Картинки, звуки (main.png тощо)
│   │       └── images/
│   ├── src/
│   │   ├── App.js
│   │   └── App.css
│   └── package.json
└── README.md
```

## Як запустити локально

### 1. Встановити залежності сервера

```bash
cd server
npm install
```

### 2. Запустити сервер

```bash
npm start
```

Сервер запуститься на **http://localhost:3001**

### 3. Встановити залежності клієнта

У **новому** терміналі:

```bash
cd client
npm install
```

### 4. Запустити клієнт

```bash
npm start
```

Клієнт відкриється на **http://localhost:3000**

### 5. Тестування (комп'ютер + телефон)

**Ведучий (комп'ютер):**
1. Відкрийте `http://localhost:3000`
2. Натисніть «Почати гру» — створиться кімната з 6-значним кодом
3. Покажіть код іншим — вони вводять його на телефоні
4. Коли гравці підключились — натисніть «Почати гру» (макс. 7 гравців)

**Гравці (телефон):**
1. Запустіть клієнт з доступом у мережі: `set HOST=0.0.0.0 && npm start` (Windows) або `HOST=0.0.0.0 npm start` (Linux/Mac)
2. Відкрийте `http://<IP-комп'ютера>:3000` у браузері на телефоні (телефон і ПК мають бути в одній мережі)
2. Натисніть «Приєднатися» → введіть код кімнати та ім'я
3. Відповідайте на питання (за правильну відповідь — 10 балів)

## Додавання нових питань

Відредагуйте файл `server/questions.js` і додайте об'єкт:

```javascript
{
  id: 4,
  text: "Текст питання?",
  options: ["Варіант 1", "Варіант 2", "Варіант 3", "Варіант 4"],
  correctIndex: 0,  // індекс правильної відповіді (0, 1, 2 або 3)
}
```

## Приклад масиву питань

```javascript
[
  {
    id: 1,
    text: "Яка столиця України?",
    options: ["Львів", "Київ", "Одеса", "Харків"],
    correctIndex: 1,
  },
  {
    id: 2,
    text: "Скільки планет у Сонячній системі?",
    options: ["7", "8", "9", "10"],
    correctIndex: 1,
  },
  {
    id: 3,
    text: "Який метал позначається символом Au?",
    options: ["Срібло", "Мідь", "Золото", "Алюміній"],
    correctIndex: 2,
  },
]
```

## Функціонал

- Підключення гравців через унікальний id сесії
- Показ питань по черзі всім гравцям
- Прийом відповідей і підрахунок балів
- Оновлення таблиці лідерів після кожного питання
- Фінальні результати після останнього питання
- Адаптивний інтерфейс для мобільних пристроїв
