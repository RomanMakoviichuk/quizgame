# Як протестувати вікторину

## 1. Запуск сервера

Відкрийте термінал і виконайте:

```bash
cd server
node server.js
```

Має з'явитися: `Сервер на http://localhost:3001`

Якщо порт 3001 зайнятий, закрийте інші процеси або змініть порт:
```bash
set PORT=3002
node server.js
```
(тоді в клієнті потрібно буде `REACT_APP_SERVER_URL=http://localhost:3002`)

---

## 2. Запуск клієнта

У **новому** терміналі:

```bash
cd client
npm start
```

Відкриється браузер на `http://localhost:3000`

---

## 3. Тест на ПК

1. У браузері має з'явитися головне меню з фоном.
2. Переконайтесь, що є текст **"Підключено ✓"** (зелений). Якщо замість цього — "Очікування з'єднання з сервером...", перевірте, що сервер запущений.
3. Натисніть **"Почати гру"**.
4. Має з'явитися екран з **4-значним кодом** (наприклад, 4827) і списком гравців.
5. Якщо нічого не відбувається:
   - Відкрийте консоль браузера (F12 → вкладка Console).
   - Подивіться, чи немає помилок (червоний текст).
   - Якщо є помилка підключення — перевірте, що сервер працює на порту 3001.

---

## 4. Тест з телефону (в одній мережі)

1. Дізнайтесь IP вашого ПК: `ipconfig` (Windows) → шукайте IPv4 (наприклад, 192.168.0.103).
2. Запустіть клієнт з доступом у мережі:
   ```bash
   cd client
   set HOST=0.0.0.0
   npm start
   ```
3. На телефоні в браузері відкрийте: `http://192.168.0.103:3000` (підставте свій IP).
4. Введіть 4-значний код з ПК і нікнейм → натисніть **"Приєднатися"**.
5. Гравець має з'явитися в списку на ПК, а на телефоні з'явиться кнопка "Почати гру" (перший гравець = адмін).

---

## 5. Тест на одному ПК (2 вкладки)

1. Відкрийте `http://localhost:3000` у першій вкладці.
2. Натисніть "Почати гру" → побачите код.
3. Змініть ширину вікна браузера до мобільної (або відкрийте DevTools → Toggle device toolbar) — має з'явитися форма "Код кімнати" і "Нікнейм".
4. Або відкрийте в інкognito/приватному режимі `http://localhost:3000` — буде та сама форма (бо ширина може визначатися як мобільна).
5. Введіть код і нікнейм → "Приєднатися".

Або простіше: відкрийте `http://localhost:3000` у звичайній вкладці — меню. У другій вкладці зменште вікно до телефонного розміру — покажеться форма приєднання. Введіть код з першої вкладки.

---

## Типові помилки

| Симптом | Причина | Рішення |
|--------|---------|---------|
| "Очікування з'єднання..." | Сервер не запущений | Запустіть `node server.js` у папці server |
| EADDRINUSE 3001 | Порт зайнятий | Закрийте інший процес або змініть PORT |
| Нічого не відбувається при "Почати гру" | Не підключено до сервера | Перевірте консоль (F12), що є "Підключено ✓" |
| Телефон не підключається | Різні мережі | ПК і телефон мають бути в одній Wi‑Fi |
